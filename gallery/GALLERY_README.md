# Project Gallery Structure

## Structure
```
/gallery/projects/[category]/[project-###]/
    ├── photos/
    │   ├── [category]-[project-###]-[photo-###].jpg
    │   └── ...
    └── videos/
        ├── [category]-[project-###]-[video-###].mp4
        └── [category]-[project-###]-[video-###].jpg (poster)
```

## Adding New Projects
1. Create new project folder: `gallery/projects/cars/project-002/`
2. Create photos/videos subfolders
3. Add media with naming convention
4. Update `script.js` SiteConfig
